### Requisitos
* Python 3.x
* pip
* docker

### Instalación 
Accede al panel de administración en `http://localhost:8000/admin/`.

```
# 1. Clona el repositorio
git clone https://github.com/usuario/proyecto-django.git
cd proyecto-django

# 2. Activa docker
sudo docker compose up --build

# 3. revisar localhost
localhost
```
